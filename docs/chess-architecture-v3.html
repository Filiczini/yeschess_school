Session-based Better Auth зберігає сесії у PostgreSQL таблиці Session. Не JWT — не потрібен Redis для інвалідації.
З коробки Email verification · Password reset · Google OAuth · Rate limiting · CSRF protection

07 — Інфраструктура на VPS
🐳
Docker Compose
Контейнеризація
web — React (Nginx static)
api — Fastify Node.js
db — PostgreSQL 16
minio — Object storage
nginx — Reverse proxy + SSL
🚀
Dokploy
Deploy Platform
Self-hosted Heroku-альтернатива.
Push to GitHub → auto deploy.
SSL через Let's Encrypt автоматично.
Env variables management.
Rollback одним кліком.
⚙️
GitHub Actions
CI Pipeline
On PR: typecheck + lint + Vitest
On merge main: build + deploy via Dokploy webhook
Secrets: через GitHub Secrets → Dokploy env
Нульові SaaS витрати — PostgreSQL, MinIO, Better Auth — все на VPS Єдині платні сервіси — Stripe (% від транзакцій) · LiqPay (% від транзакцій) · Resend (free tier)
